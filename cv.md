## Code example:
Function changes chars uppercase to lowercase and reverse.
```javascript
function lowerAndUpper(text){
    if(text.trim() !== ''){
        return Array.from(text)
        .map((char)=> char.toUpperCase() == char ? char.toLowerCase() : char.toUpperCase())
        .join("")
    } else return "Empty text!"
} 
```
